# السيرة النبوية — 46 درسًا

موقع يعرض سلسلة دروس السيرة النبوية (46 درسًا) مقسّمة على 11 مرحلة زمنية، بفهرس قابل للطي (accordion).

## الهيكل
```
seerah/
├── index.html          ← الفهرس الرئيسي (11 مرحلة، كل مرحلة تتوسع لتعرض دروسها)
└── steps/
    └── 46 ملف HTML، كل واحد بعنوان الدرس وترقيمه، مع تنقل سابق/تالي
```

## المراحل الإحدى عشرة
1. قبل البعثة والمقدمات (1–4)
2. الدعوة بمكة: السر ثم الجهر (5–11)
3. بيعتا العقبة (12–13)
4. الهجرة وقيام الدولة بالمدينة (14–18)
5. غزوة بدر الكبرى (19–23)
6. غزوة أُحد وتبعاتها (24–26)
7. غزوة الأحزاب (27–29)
8. صلح الحديبية (30–33)
9. الفتوحات الكبرى: خيبر، مؤتة، فتح مكة (34–39)
10. حنين والطائف وتبوك (40–44)
11. خاتمة السيرة (45–46)

## الحالة الحالية
- `index.html` مكتمل: فهرس بصري بهوية "مخطوطة/نور" (كريمي + ذهبي + خط Amiri)، كل مرحلة قابلة للفتح والغلق بالنقر.
- كل ملف داخل `steps/` **فارغ حاليًا (placeholder)** — فيه العنوان والترقيم وروابط تنقل (سابق/تالي/فهرس)، جاهز لإضافة المحتوى لاحقًا.

## طريقة رفعه على GitHub
```bash
git init
git add .
git commit -m "الهيكل الأساسي لسلسلة السيرة النبوية - 46 درسًا"
git branch -M main
git remote add origin https://github.com/USERNAME/seerah.git
git push -u origin main
```
لتفعيل رابط مباشر: Settings → Pages → اختار branch `main`.

## إضافة محتوى درس لاحقًا
افتح ملف الدرس، هتلاقي صندوق `placeholder` جوه `<main>` — استبدله بالمحتوى الفعلي (ممكن تستخدم آلية تلخيص المحاضرات في المشروع لو حبيت تولّد محتوى منسّق تلقائيًا)، والتصميم والتنقل هيفضلوا زي ما هم.
